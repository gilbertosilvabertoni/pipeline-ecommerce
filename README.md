## Pipeline ETL — E-commerce

Pipeline de dados construído com Python e Pandas.

### O que faz
- Extrai dados de produtos via API pública (Fake Store API)
- Transforma os dados: conversão de moeda, limpeza de texto e classificação por faixa de preço
- Salva o resultado em CSV estruturado

### Tecnologias
- Python, Pandas, Requests

### Como rodar
Abra o notebook no Google Colab e execute as células em ordem.
